Merchant

Auto merchant

The merchant loop is one of Blossom's oldest tricks — press E, read the shop name off your screen, buy what you configured, and optionally ping Discord when Mari, Jester, or Rin appears. You set the lists; Blossom does the clicking.

WinOCR required. Merchant name detection uses Windows OCR (not Tesseract). Blossom downloads the WinOCR bundle with other runtime components; you also need the Windows en-US OCR language pack. The “OCR unavailable” notice only appears when a feature actually needs OCR — not on every startup.

The basic shop flow

When auto merchant is enabled and off cooldown (~190 seconds between runs), Blossom:

  1. Presses your calibrated interact key (usually E) several times near the merchant
  2. Clicks through dialogue at merchant_dialogue_box
  3. OCR-scans for Mari, Jester, or Rin (with common misread variants)
  4. Opens the shop, buys enabled items from your per-merchant lists
  5. Optionally sends a Discord “merchant found” ping and shop screenshot

Buy lists & Set Max

On the Merchant tab (and in config), you choose which items to buy and how many. Mari and Jester use a shared Set Max calibration — Blossom clicks Max, then Purchase. Rin has no Max button, so Blossom types the amount instead.

  • Calibrate all five shop slot positions (or rely on offset from slot 1)
  • Mark Set Max once in Macro Calibrations — used for every non-Rin merchant
  • Mark Purchase and amount field if you type quantities
  • Per-merchant item toggles: mari_Items, jester_Items, rin_Items in config (managed from the UI)

Teleporter route

Some setups shop through the merchant teleporter item instead of standing at Limbo portals. Enable Auto Merchant Teleporter on the Macro tab, then use the Merchant tab's three-step banner:

Teleporter → Shop → Portable Crack

  1. Open inventory → Items tab → search “teleport” → use first slot
  2. Run the normal merchant handler at the teleporter destination
  3. Optionally return to Limbo with Portable Crack (search “crack”, amount 1, Use)

Toggle Return to Limbo after merchant teleporter on the Merchant tab if you want the crack step automatically. Calibrate inventory search and item slots under Movements / inventory calibrations.

Note: Teleporter and Limbo merchant features are disabled while Fishing mode is active — fishing owns the session until you turn it off.

Limbo merchant

Auto Merchant in Limbo runs the shop loop when you are already in Limbo without the teleporter prelude. Useful for standard portal camping setups.

Discord pings

When a merchant is found, Blossom can post to your webhook URL with an embed and your private server rejoin link. Optional shop screenshot webhooks fire after purchases. Configure URLs and per-merchant ping roles on the Webhook tab — webhooks guide.

Tips for reliable OCR

  • Fullscreen Roblox at fixed resolution — same as calibration day
  • Use Arial or Rubik UI font in Roblox
  • Mouse delay 1000–2000 ms if names are misread or clicks double-fire
  • Enable OCR failsafe on Macro tab for backup regions

If merchant runs but buys the wrong slot, recalibrate all five positions — do not rely on pixel offsets alone after UI scale changes.
