Macro for Sol's RNG — biomes, auras, merchants, potions, fishing. Free on stable releases.

Fork of the Coteab macro by Vapure, built on the original Noteab biome macro. Blossom adds its own UI and Python backend.

What Blossom does

  • Log-based biome and aura detection
  • Auto merchant — OCR for Mari / Jester / Rin, optional teleporter and buy lists
  • Auto potions, fishing mode, anti-AFK
  • Discord webhooks — rare biomes and merchants

Changelog lives in the app Notice tab and on GitHub Releases.
