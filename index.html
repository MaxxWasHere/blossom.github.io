Blossom is a fork of the Coteab macro by Vapure, built on the original Noteab biome macro. Same macro DNA — Blossom adds its own sakura UI, Python backend, calibrations hub, reconnect flow, and Custom UI theming.

What Blossom does

  • Biomes & auras — log-based detection once calibrated; session stats in the side panel
  • Auto merchant — WinOCR finds Mari / Jester / Rin, buys from your lists, optional teleporter route and return to Limbo via Portable Crack
  • Auto potions — crafts and rotates potions in any biome, including GLITCHED, DREAMSPACE, CYBERSPACE
  • Fishing mode — bite detection, reel minigame, walk to dock, optional sell at the fish shop; failsafe rejoin when stuck
  • Auto Reconnect — relaunch Roblox, join your private server, click calibrated Reconnect Start; forced sell after reconnect when inventory was full
  • Discord webhooks — rare biome and merchant pings with your private server link
  • Anti-AFK — subtle mouse activity so long sessions stay alive

The in-app Notice tab has the full changelog. Stable release notes also appear on GitHub Releases.
