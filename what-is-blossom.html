What is Blossom?

What is Blossom?

Blossom runs Sol's RNG for you — rolling biomes, buying merchants, brewing potions, and fishing — while you are AFK. Think of it as a patient co-pilot: you calibrate once, flip on what you need, and it keeps clicking while you live your life.

Lineage: Noteab (NotWindyZ) → Coteab (Vapure) → Blossom (MaxxWasHere). Same macro goals; Blossom ships its own UI, backend, releases, and Discord integrations.

What it does

  • Biomes & auras — reads Roblox logs for reliable detection after calibration; optional Discord pings for rare rolls
  • Auto merchant — WinOCR finds Mari, Jester, or Rin; buys from your lists; optional teleporter route; can return to Limbo with Portable Crack after shopping
  • Auto potions — crafts and rotates potions in any biome, including GLITCHED, DREAMSPACE, and CYBERSPACE
  • Fishing mode — detect bite, play reel minigame, walk to dock (recorded path or default), optional sell trip; failsafe rejoin when stuck at the dock
  • Auto Reconnect — if Roblox closes mid-macro, terminate client, rejoin private server, click calibrated Reconnect Start; forced sell on next fishing cycle after reconnect
  • Discord webhooks — biome notifier and merchant pings with embeds and your private server rejoin link
  • Anti-AFK — light mouse movement and session statistics

How it is built

Blossom combines a React UI in WebView2 with a Python automation backend (screen clicks, log tailing, OCR). Fixed-name launchers Blossom.exe and Blossom-beta.exe bootstrap the versioned app payload and sync optional OpenCV / WinOCR bundles.

You customize behavior through calibrations (screen positions), config toggles, recorded walk paths, and — if you like — Custom UI CSS for the app chrome. Full tour: Features & tabs.

Get it

  • Stable (free): MaxxWasHere/blossom releases
  • Beta (keyed): MaxxWasHere/blossombeta releases

Setup in four steps

  1. Run the launcher and complete beta activation if needed
  2. Calibrate in Macro Calibrations — video tutorial
  3. Enable modules on the Macro tab and set your private server link in Misc
  4. Optional: webhooks on the Webhook tab, theme on Appearance

Advanced: record custom walk paths with tools/blossom_path_recorder.py in the source repo. Ship paths in paths/fishing.json and paths/char_align.json for replay during fishing prep.

FAQ

Is it a virus?

No. Source is public on GitHub. Windows Defender often flags PyInstaller-packed exes — scan on VirusTotal or run from source if you prefer.

Black or white window?

Install WebView2 Runtime.

Misdetects items or wrong potions?

Recalibrate OCR regions or raise mouse delay to 1000–2000 ms. Arial or Rubik font in Roblox helps WinOCR.

Stable vs beta?

Stable is free with no key. Beta gets newer features first but requires a Discord license key and HWID binding. Settings guide

Where do I put walk paths and themes?

Under %LOCALAPPDATA%\Blossom\paths\ and themes\. See Settings & config.

Can I customize the look?

Yes — eight built-in themes plus full Custom UI CSS. Theming guide
