Getting Started

Getting Started

Install Blossom, calibrate once, then turn on the automations you want.

Install

  1. Download Blossom.exe from stable releases (or Blossom-beta.exe for beta).
  2. Run it once. The bootstrapper syncs the full app into %LOCALAPPDATA%\Blossom\.
  3. Keep using the same launcher — it checks for updates on each run.

Beta builds show an activation screen on first launch. Get a key from the Discord bot (/getkey).

First-time setup

  1. Calibrate — open Macro Calibrations and follow Calibrations or this video.
  2. Macro tab — turn on merchant, potions, fishing, and other modules you want.
  3. Webhook tab (optional) — paste a Discord webhook URL for biome or merchant pings.
  4. Appearance (optional) — pick a built-in theme or Custom UI CSS (Theming).

Config location

Settings, paths, potion files, and custom themes live under:

%LOCALAPPDATA%\Blossom\

Run the macro

Use Start in the header bar (or your configured hotkey). Run Blossom as administrator if clicks do not register in Roblox.
