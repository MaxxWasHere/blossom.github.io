Theming

Blossom supports built-in themes and Custom UI (your own CSS).

Built-in themes

Appearance tab → pick Pink, Dark, Light, OLED, Sakura, Midnight, Forest, etc. Choice is saved in config.

  • UI scale and window size (Small / Medium / Large / Custom)
  • Reduce motion — disables heavy animations
  • Always on top (Settings)

Custom UI (CSS)

  1. Open %LOCALAPPDATA%\Blossom\themes\
  2. Copy custom-ui.example.css → my-theme.css
  3. Edit CSS (tokens and selectors — the example file is fully commented)
  4. Appearance → Custom UI → select your file → Refresh

Rules

  • CSS only — no HTML, no JavaScript, no @import, no remote url()
  • Custom UI overrides built-in theme colors when active
  • Cannot add new tabs or macro features — styling only

Design tokens (common)

TokenPurpose
--accentPrimary accent color
--bg-root, --bg-sidebar, --bg-cardBackgrounds
--text-primary, --text-secondary, --text-mutedText
--border, --radius-mdBorders and corners

Full list and DOM selectors: open custom-ui.example.css in your themes folder (created on first Custom UI use).

HTML hints for advanced skins

  • html.blsm-custom-ui-active — Custom UI is on
  • html.blsm-light-ui — light palette hint
  • html.blsm-reduce-motion — user disabled motion

Sharing themes

Publish your .css on Discord or GitHub. Others drop it into their themes folder and select it in Appearance.
