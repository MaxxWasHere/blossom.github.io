Blossom

An independent fork of the Noteab / Coteab macro for Sol's RNG. Same core goals — biomes, merchants, potions, fishing — with Blossom's own sakura UI, Python backend, bootstrap launchers, and Discord integrations.

What Blossom adds

  • Sakura branding, sidebar layout, Appearance tab with eight built-in themes and Custom UI CSS
  • Macro Calibrations hub, fishing mode, Auto Reconnect, merchant Limbo return, prep/fishing failsafes
  • Potion crafting, rotation, and Start / Stop hotkeys
  • Config under %LOCALAPPDATA%\Blossom\ — settings, paths, themes
  • Discord webhooks and biome notifier from the Python backend
  • Fixed-name bootstrap launchers, built-in updater, beta licensing with HWID
  • WinOCR for merchant/Eden/biome OCR; OpenCV for fishing vision
  • Command palette (Ctrl+K), launch overlay, runtime component sync

Lineage (upstream macros)

1. Noteab biome macro — NotWindyZ

Earliest public biome-macro foundation. Apache 2.0. github.com/NotWindyZ/Noteab-Macro

2. Coteab macro — Vapure (xVapure)

The macro most players ran before Blossom. Direct basis for merchant OCR, log detection, and calibrations. github.com/xVapure/Noteab-Macro

3. Blossom — MaxxWasHere

This fork. UI and backend maintained separately with its own release channels. github.com/MaxxWasHere/blossom
