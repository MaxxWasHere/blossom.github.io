Blossom

An independent fork of the Noteab / Coteab macro for Sol's RNG. Same core goals — biomes, merchants, potions, fishing — with Blossom's own UI, Python backend, releases, and Discord integrations.

What Blossom adds

  • Sakura branding, sidebar layout, and Appearance / Custom UI theming
  • Macro Calibrations hub, fishing mode, reconnect flow, and merchant Limbo return
  • Potion crafting, rotation, and Start / Stop hotkeys
  • Config under %LOCALAPPDATA%\Blossom\
  • Discord webhooks and biome notifier from the Python backend
  • Built-in updater, beta licensing, and bootstrap launcher (Blossom.exe)

Lineage (upstream macros)

1. Noteab biome macro — NotWindyZ

Earliest public biome-macro foundation. Apache 2.0. github.com/NotWindyZ/Noteab-Macro

2. Coteab macro — Vapure (xVapure)

The macro most players run today. Direct basis for Blossom. github.com/xVapure/Noteab-Macro

3. Blossom — MaxxWasHere

This fork. UI and backend maintained separately. github.com/MaxxWasHere/blossom
