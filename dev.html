Local development

Prerequisites: Python 3.14+, WebView2 runtime, pip install -r requirements.txt

py run_local_ui.py

Custom UI template lives in src/blossom_custom_ui.py (ensure_sample_custom_ui_file()). Appearance logic: assets/blossom-appearance.js. Built-in theme tokens: assets/blossom-themes.css.

Release build

Close running Blossom instances first. Output separates launchers (launchers/) from versioned payload exes (app/).

py scripts\build_all.py all --stable-version 2.2.4 --beta-version 2.2.4-beta.1 -o dist\2.2.4 --bootstrap

Version bump checklist

  1. src/blossom_build_info.py → APP_VERSION
  2. scripts/build_all.py default versions
  3. src/blossom_updater.py if release URLs change
  4. assets/noticetabcontents.txt — user-facing changelog (stable heading only, no beta suffix in title)
  5. Tag GitHub release; upload launchers + payload; mirror beta notes to blossombeta repo if applicable

Custom UI sanitizer — src/blossom_custom_ui.py blocks @import, remote url(), script-like CSS, and files over 512 KB. Document changes in theming docs when selector maps change.
