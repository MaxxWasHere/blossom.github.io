Calibrations

Blossom automates by reading your screen and game logs. Calibrations tell it where to click — without them, merchant OCR, fishing, inventory actions, and reconnect will miss.

Watch video tutorial

Match the same Roblox resolution, window mode, and UI scale every time you run the macro. If you change display settings, recalibrate or import a preset for your new setup.

Macro Calibrations hub

Open Macro Calibrations in the sidebar. The unified hub (#blsm-cal-hub) groups calibrations by feature instead of scattered legacy accordions — older panels stay hidden while the hub is active.

Each category has tabs for mark points (click positions), steps, and hints. Use the mouse calibration card to capture coordinates while Roblox is focused.

General tips

  • Fullscreen Roblox at a fixed resolution is the most reliable setup.
  • Set mouse delay to 1000–2000 ms if OCR mis-clicks; lower only when everything is stable.
  • Use Arial or Rubik font in Roblox — WinOCR reads them more reliably.
  • Community presets — import macro_calibs_preset.json from upstream when your resolution/scale matches; otherwise calibrate manually once per PC.
  • Collections and UI clicks during fishing prep now jump to the button, settle, then click — still calibrate accurate positions.

Merchant teleporter & Limbo

The Merchant tab shows a three-step flow: Teleporter → Shop → Portable Crack. After a teleporter shopping run, Blossom can use Portable Crack from inventory (search “crack”, amount 1, Use) to return to Limbo. Toggle under Merchant → Return to Limbo after merchant teleporter.

OCR & runtime

Merchant, Eden, and biome-selector OCR use WinOCR (not Tesseract). Install the Windows OCR en-US language pack if prompted. The OCR-unavailable notice only appears when a feature that needs OCR runs without WinOCR — not on every startup.

Import / export

Share your calibration JSON with friends on the same resolution, or back up before reinstalling. Config is preserved across Blossom app updates — only game UI moves force recalibration.

After updating Blossom or the game

Re-check calibrations if a Roblox update moves UI elements. Your config file survives app updates; you usually only re-mark moved buttons.
