Calibrations

Blossom reads screen positions and game logs. Calibrations are required for merchant OCR, fishing, inventory actions, and reconnect.

Watch video tutorial

Macro Calibrations hub

Open Macro Calibrations in the sidebar. The unified hub groups calibrations by feature instead of scattered legacy panels.

Tips

  • Match Roblox resolution and UI scale to what you calibrated (fullscreen recommended).
  • Set mouse delay to 1000–2000 ms if OCR mis-clicks; Arial or Rubik font in Roblox helps OCR.
  • Use presets when your resolution/scale/mode matches a community preset (from upstream macro_calibs_preset.json).

Common calibration groups

FeatureWhat to calibrate
MerchantShop regions, talk/buy clicks, OCR failsafe
FishingRod, reel minigame, dock walk, sell
Inventory / BR / SCSearch bar, item slots (Portable Crack / Limbo return)
ReconnectStart button after Roblox relaunch, private server link in Settings

Import a full preset JSON from GitHub if you share configs with your resolution — otherwise calibrate manually once per PC.

After updating Blossom

Re-check calibrations if a game update moves UI elements. Your config file is preserved across app updates.
