Troubleshooting

Common fixes when Blossom won't start, mis-clicks, or misreads the screen.

App won't start or window is blank

  • Install Microsoft WebView2.
  • Run Blossom.exe from releases — not only the inner Blossom-x.y.z.exe unless you know you need it.
  • Try running as administrator.
  • Full reinstall: delete %LOCALAPPDATA%\Blossom\ only if you have a config backup.

Antivirus / "virus" warning

Blossom is open source. Windows Defender often flags PyInstaller exes. Scan on VirusTotal or run in a VM if unsure.

Macro idle or clicks miss

  • Run Blossom as administrator.
  • Keep Roblox focused; match calibrated resolution and UI scale.
  • Increase mouse delay (1000–2000 ms).
  • Recalibrate OCR failsafe — see Calibrations.

Wrong potions / OCR misreads

  • Recalibrate merchant / OCR regions.
  • Use Arial or Rubik font in Roblox settings.
  • Raise input delay.

Beta: license errors

  • Check internet and firewall.
  • Use /resethwid in Discord once if you changed PCs.
  • Beta keys are one device per key.

Updates

Use the launcher Blossom.exe — it checks GitHub for your channel. If an update fails, download manually from Releases.
